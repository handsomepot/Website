Tutorial

This tutorial is based on Riley Chen's website and will teach you some animation effects behind it.
I hope you will enjoy this tutorial and I hope it could be useful.


I. progress bar

50%
75%
100%

1. create background container

HTML

<div id ="block_container">
</div>

CSS

#block_container {
    display: flex;
    width:500px;
    height:20px;
    background-color:lightgrey;
    margin:25px auto;
    border-radius:25px;
}


Here we create a bar with lightgrey background

2. create the bar

HTML

<div id ="block_container">
  <div class="block1">50%</div>
</div>

CSS

.block1{
   width:50%;
   background-color:green;
   border-radius:25px;
   text-align:center;
}


Then we add another div into our block-container and color it as green.
Change the width of a progress bar with the CSS width property (from 0 to 100%).

2. animate the bar

CSS

.block1{
   width:50%;
   background-color:green;
   border-radius:25px;
   text-align:center;
   animation-timing-function: linear;
   animation-fill-mode: forwards;
   animation-name:progress1;
   animation-duration: 1s;

}

@keyframes progress1 {
  from{
        width:0;
    }
    to{
        width:50%;
    }
}
