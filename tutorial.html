1.create a box including an icon, a header and text

HTML

<!DOCTYPE html>
<html>

<head>
    <title>Font Awesome Icons</title>
    <meta name="viewport" content="width=device-width, initial-scale=1">
    <link rel="stylesheet" href="https://cdnjs.cloudflare.com/ajax/libs/font-awesome/4.7.0/css/font-awesome.min.css">
</head>

<body>

    <section class="box" id="flip1">
        <i id="icon" class="fa fa-user"></i>
        <h3>Flip1</h3>
        <p>This is the first animation</p>
    </section>

</body>

</html>

CSS

#icon{
     font-size:25px;
     background-color:#3cadd4;
     color: #ffffff;
     padding:28px;
     border-radius:50%;
}

.box {
     border-radius: 4px;
     border: solid 1px rgba(144, 144, 144, 0.25);
     margin-bottom: 2em;
     padding: 1.5em;
     width:25%;
}


2. add hover effects


When the box is hovered, we swap the background color and icon color.

CSS

#flip1:hover{
  background-color: #3cadd4;
  color:#ffffff;
}

#flip1:hover i{
  background-color: #ffffff;
  color:#3cadd4;
}

#flip1:hover h3{
  color:#ffffff;
}



 

Change Color

This is what it looks like


hover to play


3. flip the icon

We now play the flip animation when hovered.

CSS

#flip1 i{ 
 transition: transform 0.5s ease-out;
  
}

#flip1:hover i{
  background-color: #ffffff;
  color:#3cadd4;
  transform: rotateY(180deg);

}

II. progress bar

50%
75%
100%
Hover to animate.

1. create background container

HTML

<div id ="block_container">
</div>

CSS

#block_container {
    display: flex;
    width:500px;
    height:20px;
    background-color:lightgrey;
    margin:25px auto;
    border-radius:25px;
}


Here we create a bar with lightgrey background

2. create the bar

HTML

<div id ="block_container">
  <div class="block1">50%</div>
</div>

CSS

.block1{
   width:50%;
   background-color:green;
   border-radius:25px;
   text-align:center;
}


Then we add another div into our block-container and color it as green.
Change the width of a progress bar with the CSS width property (from 0 to 100%).

2. create the bar

CSS

.block1{
   width:50%;
   background-color:green;
   border-radius:25px;
   text-align:center;
   animation-timing-function: linear;
   animation-fill-mode: forwards;
   animation-name:progress1;
   animation-duration: 1s;

}

@keyframes progress1 {
  from{
        width:0;
    }
    to{
        width:50%;
    }
}


Then we add another div into our block-container and color it as green.
Change the width of a progress bar with the CSS width property (from 0 to 100%).
