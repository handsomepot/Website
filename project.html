1. Phaser 3 template

Phaser is a fast, free and fun open source framework for Canvas and WebGL powered based games.

The benefit of having a template is that it saves you a lot of time when you are setting up your game.

This template is reusable and it includes thes following scenes:

  • Preloader – loads all the assets that are needed, displays a logo, and a progress bar.
  • Menu – the title screen of the game.
  • Game – contains the main logic of the game.
